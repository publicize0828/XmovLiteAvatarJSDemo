# 更新日志

本文档记录项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.2] - 2025-12-25

### 新增

- **对话状态管理器** (`src/services/action-manager.ts`)
  - 引入 `ActionManager`，统一处理 speak 队列、流式片段排队与首尾态标记
  - 支持在多段内容连续播报时自动排队，避免上下文被打断

### 优化

- **数字人状态管理** (`src/stores/app.ts`)
  - `sendMessage` 接入 `ActionManager`，首帧/末帧由状态管理器自动控制
  - 断开连接时自动重置队列，防止悬挂任务
  - `avatarState` 在语音开始/结束时即时刷新，前端状态栏更准确
- **渲染容器体验** (`src/components/AvatarRender.vue`)
  - 桌面端默认 9:16 视窗，并确保 SDK 内部节点居中
  - 移动端横竖屏自适应：竖屏下按 9:16 填充可视区域（可根据需要开启或调整）

### 修复

- 修复某些场景下连续调用 `speak` 导致的状态错乱问题
- 修复虚拟人实例在断开后仍残留引用导致的二次连接异常

## [1.0.1] - 2025-01-XX

### 新增

- **SSE 流式解析器** (`src/utils/sse-parser.ts`)
  - 新增 `SSEParser` 类，用于解析 Server-Sent Events (SSE) 流式响应
  - 支持异步迭代器模式，兼容不同浏览器环境
  - 提供 `createChunkIterator` 方法，将 Response 转换为异步可迭代对象

### 优化

- **SSML 生成函数优化** (`src/utils/index.ts`)
  - 添加特殊字符自动转义功能（`<`, `>`, `'`, `"`, `&`），提升 SSML 输出的安全性和兼容性
  - 优化换行符处理，将多个连续换行符统一为一个，提升语音播报的自然度

- **流式对话处理逻辑重构** (`src/stores/app.ts`)
  - 采用更智能的文本缓存和切分策略：
    - 最小字符数阈值（minimum = 20）控制首句发送时机
    - 区分中文和英文标点符号处理（`cnSplitSign` / `enSplitSign`）
    - 统计可读字符数（汉字、英文字母、数字）
    - 处理英文空格计数，优化英文文本的切分逻辑
  - 优化流式文本切分算法：
    - 首句需要达到最小字符数且遇到标点符号才发送
    - 后续句子遇到标点符号即可发送，提升响应速度
    - 更自然的语音播报节奏和流畅度

- **性能优化**
  - 移除 `think()` 调用，减少不必要的延迟，提升对话响应速度
  - 优化代码结构，移除未使用的导入和函数

### 修复

- **SSE 解析器兼容性修复** (`src/utils/sse-parser.ts`)
  - 修复 `ReadableStream` 异步迭代器兼容性问题
  - 统一使用 `getReader()` 方法读取流，确保在所有浏览器环境下正常工作
  - 添加资源释放逻辑（`reader.releaseLock()`），防止内存泄漏

### 技术改进

- 代码质量提升：修复所有 TypeScript lint 警告
- 更好的类型安全：完善类型定义和错误处理
- 改进的错误处理：添加防御性编程检查

## [0.0.0] - 初始版本

初始项目发布，包含以下核心功能：

- 虚拟人实时渲染（XmovAvatar SDK 集成）
- 语音识别链路（腾讯云 ASR 集成）
- LLM 流式播报（豆包大语言模型集成）
- 字幕与动画反馈
- 一体化配置面板
- SDK 动态加载机制
